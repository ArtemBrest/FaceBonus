Slot Features

The Grimoire is the Wild symbol in the game that acts both as Wild and as a Scatter. As a Wild, it can substitute for all the symbols in the game to help form winning combos, and as a Scatter, it will trigger the bonus round.

Splitting Symbols

This is a randomly activated feature that concerns and is applicable only to the three middle reels, however, it can be activated in the base game and in the bonus round as well, and once it does, the active reels will be split into two and increasing the number of win ways.

Free Spins
